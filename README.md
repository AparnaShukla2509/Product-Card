#  AI Product Content Generator

A **React.js web application** that generates **AI-powered product details** (title, description, and tags) based on user input and displays them in a beautifully styled product card.

##  Features
- **Product Form**: Enter product name and select category
- **AI Content Generation**: Generates creative titles, descriptions, and relevant tags
- **Styled Product Card**: Displays generated content in a professional card layout
- **Responsive Design**: Works seamlessly on desktop and mobile devices

##  Tech Stack
| Technology | Purpose |
|------------|---------|
| React.js 18 | Frontend framework |
| Pure CSS | Styling with modern UI effects |
| Axios | REST API calls |

# Set up
cd ai-product-card

# Install dependencies
npm install

# Start development server
npm start


Open browser: **http://localhost:3000**



##  Available Scripts

| Command | Description |
|--------|-------------|
| `npm start` | Run development server |
| `npm test` | Launch test runner |
| `npm run build` | Build for production |


## How to Use

1. Enter **Product Name** (e.g., Coffee Mug)
2. Select **Category** (e.g., Home & Kitchen)
3. Click **Generate Details**
4. View AI-generated title, description, and tags
